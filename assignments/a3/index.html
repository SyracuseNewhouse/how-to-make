Assignment 3: APIs

Instructions

Use Google to find an API. Use a wireframing service OR pencil and paper to design an application that would use it. Explain which API calls you are using, what information you are passing in, and what you are getting back. If possible, provide examples of real API responses. (Due 4/14 by noon)

To submit: email your project to htmao-s14@googlegroups.com along with your wireframes, and a short description of the "API endpoints" you are using.

Tools

  • Google
  • JSON Beautifier
  • Balsamiq (has a 7-day free trial)
  • Mockflow (freemium)
