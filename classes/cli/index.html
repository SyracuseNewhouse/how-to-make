Command Line Interface

Terms

  • terminal
  • ssh
  • directory

Commands

  • pwd - shows the location of the current directory.
  • ls - lists the content of the current directory.
  • cd {PATH} (e.g. "cd /var/www") - changes the current directory.
  • cat {PATH} (e.g. "cat README") - shows the content of a file.
  • man {COMMAND} (e.g. "man ls") - provides an explanation of a command.
  • ssh {COMPUTER IP ADDRESS} - connects to the command line of another computer.
